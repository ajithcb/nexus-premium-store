# Nexus Premium - Full-Stack E-Commerce Platform

Live Frontend Shop: https://onrender.com
Live Backend API Node: https://onrender.com

Welcome to Nexus Premium, a responsive, full-stack e-commerce marketplace built using the MERN stack. The platform serves a dynamic inventory directly from a cloud database cluster and features robust session authentication alongside real-time search capabilities.

## Architecture & Features
* **Live Search Optimization:** Features a dynamic text-filter bar that updates the product catalog view instantaneously.
* **State Preservation Loop:** Integrates secure local storage hooks to manage user profiles and shopping baskets through page reloads.
* **Secure Session Models:** Protects user data and pipeline routes using password encryption and token verification.
* **Unified API Cloud Infrastructure:** Hosts server scripts and client layouts independently on Render's cloud platform.

## Complete Technology Stack
* **Frontend:** React.js, Vite, React Router DOM, Styles
* **Backend:** Node.js, Express.js, REST APIs
* **Database:** MongoDB Atlas, Mongoose ODM
* **Security:** JSON Web Tokens (JWT), Bcrypt.js, LocalStorage API
* **Version Control & DevOps:** Git, GitHub, Render Platform

## Production Deployment Layout
The workspace is structured at the main root repository layer for seamless cloud compilation:
* `/server.js` - Main self-contained backend API engine.
* `/App.jsx` - Main self-contained React frontend interface.
* `/index.html` - Base mounting webpage canvas template.
* `/package.json` - Node package dependency registry card.
